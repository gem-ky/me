# Tunime (redesign) — свой каталог аниме

Переосмысление an0ncer.github.io/Tunime: своя реализация, свой дизайн (тёмная готическая эстетика),
без копирования чужого кода. Каталог берётся из открытого Shikimori API.

## Структура
- `backend/` — сервер на Express: регистрация/вход, личный список просмотра, прокси к Shikimori API
- `frontend/` — статические страницы (можно выложить на GitHub Pages, как оригинал)

## Как запустить локально

### 1. Бэкенд
```
cd backend
npm install
npm start
```
Сервер поднимется на http://localhost:3000.
Данные хранятся в файле `backend/data.json` (создастся автоматически при первом запуске) —
никакой отдельной установки базы данных не нужно.

Перед реальным использованием поменяй `JWT_SECRET` в `backend/.env` на случайную строку.

### 2. Фронтенд
Просто открой `frontend/index.html` в браузере, либо подними локальный сервер:
```
cd frontend
python3 -m http.server 8080
```
и зайди на http://localhost:8080

Важно: в `frontend/app.js` есть константа `API_BASE` — сейчас она указывает на localhost,
после деплоя бэкенда на хостинг поменяй её на реальный адрес.

## Деплой (бесплатно)
- **Фронтенд** → GitHub Pages (как у оригинала: Settings → Pages → выбрать ветку)
- **Бэкенд** → Render.com или Railway.app, бесплатный тариф. Просто подключаешь репозиторий,
  указываешь Root Directory = `backend`, Build Command = `npm install`, Start Command = `npm start`
- Не забудь задать переменную окружения `JWT_SECRET` на хостинге
